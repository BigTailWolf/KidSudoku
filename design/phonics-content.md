## Audio Sourcing Plan

### Priority 1: Public Domain / Open Resources
- Search for openly licensed Phonics audio packs
- Possible sources: LibriVox, Forvo, Open Speech Repository
- Check Creative Commons licensed educational resources

### Priority 2: TTS Generation
- Use high-quality child-friendly TTS voice
- ElevenLabs or Google Cloud TTS (child voice if available)
- Generate all 168 clips programmatically
- Manual QA for pronunciation accuracy

## Illustration Style Guide

- **Style:** Flat, colorful vector illustrations
- **Palette:** Bright primary colors + pastels
- **Characters:** Cute, round, friendly — suitable for 3-6 year olds
- **Consistency:** All 126 cards share the same art style
- **Generation:** AI-generated (Midjourney/DALL-E/Flux) with consistent style prompt
- **Format:** PNG with transparent background, 512×512 or 1024×1024
- **Card frame:** Uniform template with word text + phoneme tag at bottom
