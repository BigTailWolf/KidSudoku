# Gacha (抽卡) System Design

## Overview

每完成一关数独，奖励一次抽卡机会。抽卡从当前 Collection（Phonics Group）的 18 张卡池中随机抽取，可重复。

---

## Parameters

| 参数 | 值 |
|---|---|
| 每个 Collection 卡池 | 18 张 |
| 每关奖励 | 1 次抽卡 |
| 是否可重复 | ✅ 是 |
| 关卡数量 | **无限**（动态生成） |
| 保底机制 | 无（纯随机） |

## Probability

- 均匀分布：每张卡 1/18 ≈ 5.56% 概率
- 不设稀有度分级（所有卡片同等概率）
- 统计期望：集齐 18 张约需 ~60 次抽卡（优惠券收集问题）
- 无关卡限制，孩子可以一直玩一直抽

## Infinite Play

- 每次点击 "Play" 动态生成全新数独题目
- 完成即奖励 1 次抽卡
- 无需"重玩"概念，每一局都是新的
- 这保证了：
  - 无限练习量
  - 总能集齐卡片
  - 永远不会遇到重复题目

## Duplicate Handling

- 重复卡仍然展示完整翻卡动画
- 仍然播放单词发音（额外的学习机会）
- 显示 "Already collected!" 标记
- 不给予额外奖励（无合成/升级机制，保持简单）

## Emotional Design

| 场景 | 体验设计 |
|---|---|
| 新卡 | 🎉 大特效，金光，"New!" 标签 |
| 重复卡 | ✨ 温和特效，仍然正面鼓励 |
| 差一张集齐 | 展柜页显示进度，制造期待感 |
| 集齐 | 🎊 全屏庆祝，解锁成就感 |

---

## Anti-Frustration

由于目标用户是 3-6 岁儿童：
- **没有惩罚机制** — 重复卡不会让孩子沮丧
- **每次抽卡都有发音学习** — 即使重复也有教育价值
- **进度只增不减** — 不会丢失已收集的卡片
- **无限重玩** — 永远有机会集齐
