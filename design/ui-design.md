## Screen Details

### 1. Splash Screen
- App logo + cute animation
- Auto-transition to Home (2 seconds)

### 2. Home Screen (World Map)
- 7 个 Collection 显示为地图上的"岛屿"或"星球"
- 每个岛屿显示：Collection 名称 + 收集进度 (如 5/18)
- Group 1 默认解锁，后续 Group 需完成前一组 50% 关卡才解锁
- 右上角：展柜入口按钮 🏆
- 整体风格：探险地图，色彩丰富

### 3. Collection Detail
- 显示该 Group 的所有关卡（网格排列）
- 每个关卡显示：编号 + 状态（🔒未解锁 / ⭐已完成 / 空=可玩）
- 关卡线性解锁（完成第 N 关解锁第 N+1 关）
- 底部预览：该 Collection 的卡片缩略图（已收集的显示，未收集的灰色剪影）
- 每个 Collection 包含关卡数量：**30 关**（确保足够抽卡次数集齐 18 张）

### 4. Sudoku Game Screen
- 顶部：计时器 ⏱ + 重置按钮 🔄
- 中间：数独网格（4×4 或 6×6）
  - 预填数字：深色，不可修改
  - 玩家填入：浅色/彩色，可修改
  - 错误提示：无实时提示（提交时检查）
- 底部：数字选择键盘
  - 4×4 模式：数字 1-4
  - 6×6 模式：数字 1-6
  - 橡皮擦按钮（清除已填数字）
- 填满所有格子 → 自动检查
  - ✅ 正确：庆祝动画 → 跳转抽卡
  - ❌ 错误：高亮冲突格子，继续修改

### 5. Gacha (抽卡) Screen
- 展示一张翻转的卡片（背面朝上）
- 孩子点击/滑动 → 翻卡动画
- 翻转后显示卡片正面：
  - 卡通插画
  - 单词（大字）
  - 音素标注（小字）
  - 自动播放发音
- 如果是重复卡：显示 "Already collected! ⭐" 但仍播放发音
- 如果是新卡：显示 "New! 🎉" + 星星特效
- 底部按钮：
  - "听一遍" 🔊（重播发音）
  - "继续" →（返回关卡列表）

### 6. Showcase (展柜) Screen
- 顶部：Collection 切换 Tab（Group 1-7）
- 网格展示 18 张卡片位置：
  - 已收集：显示完整卡片，可点击听发音
  - 未收集：灰色剪影 + "?"
- 底部：收集进度条 + 数字 (如 12/18)
- 集齐动画：所有卡片发光 + 彩带 🎊

## Interaction Patterns

### 数字输入
1. 点击空格子 → 格子高亮
2. 点击底部数字 → 填入
3. 再次点击已填格子 → 可修改
4. 点击橡皮擦 → 清除选中格子

### 适配 Kindle Fire
- **最小触摸目标：48dp**（小手友好）
- **无需多指手势**（单点触控即可）
- **横屏/竖屏：竖屏锁定**（更适合 Kindle 使用场景）
- **Kindle Fire 分辨率适配：**
  - Fire 7: 1024×600
  - Fire HD 8: 1280×800
  - Fire HD 10: 1920×1200

## Typography

- **主字体:** 圆润无衬线体（如 Nunito, Baloo, Comic Neue）
- **数独数字:** 清晰等宽字体，加大字号
- **单词显示:** 粗体，大字号，便于辨认
- **音素标注:** 斜体，彩色高亮目标音素
